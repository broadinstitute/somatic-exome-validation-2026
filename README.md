# Clinical Validation of Somatic Exome Sequencing Assay

## Reference

This repository contains the code used to generate analyses and figures presented in:

"Clinical Validation of a High-Performance Somatic Exome Sequencing Assay: From Target-Enrichment Strategy to Variant Calling"

Tsuji, J., and Rickles-Young, M., et al (2026) npj Genomic Medicine.


## Analysis and Figures

All necessary code and data are in each subdirectory.
**Since Figure 1 was generated with illustration tools such as PowerPoint and BioRender, the subdirectories start from Figure 2.**
